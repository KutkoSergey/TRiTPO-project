Поток событий для прецедента "Промотр статистики"
Краткое описание:
Данный вариант использования позволяет пользователю просмотреть статистику по играм, 
а именно счет и ник игрока.
Предусловия:
1. Пользователь должен запустить приложение.
2. Пользователь должен нажать на кнопку "Statistics"
Основной поток событий:
1. Система отображает статистику игры, а именно ник и количество очков, набранных за игру.
2. Пользователь после просмотра может вернуться обратно в главное меню приложения.
Пост условия:
1. Переход в главное меню приложения.

Поток событий для прецедента "Начать игру"

Данный вариант использования позволяет выбрать режим игры.
Предусловия:
1. Пользователь должен запустить приложение.
2. Пользователь должен нажать на кнопку "Start Game"
Основной поток событий:
1. Система отображает меню с режимами игры.
2. Пользователь  может вернуться обратно главное меню приложения.
3. Пользователь может выбрать режим игры.

2. Запуск игровой локации.

Поток событий для прецедента "Изменить положение игровой фигуры".
Краткое описание:
Данный вариант использования игроку во время игры изменить положение .
Предусловия:
1. Пользователь должен во время игры с помощью нажатия клавиши может
изменить положение игровой фигуры.
Основной поток событий:
1. После нажатия клавиши фигура подает запрос полю на изменения положения.
2. Если столкновений нету, то есть поле дает положительный ответ, то игровая
фигура изменяет свое положение.

Пост условия:
1. Игровая фигура изменяет свое положение.
